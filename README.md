# Dear Movie Diary - a favorite movies app.

## Prerequisites
- An OMDB API Key: https://www.omdbapi.com/

## Run App
1. clone repo
1. `cd dear-movie-diary`
1. `cp .env.example .env`
1. edit .env and add your `OMDB_API_KEY`
1. `npm install`
1. `npm start`

A browser window should open at `localhost:8080`
